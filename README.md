此项目运行编译为手动运行,点击Actions > 编译X86_64_lede > Run workflow 
 
    直接编译点击绿色Run workflow，需要在线编译修改false为true
    
    需要SSH在线编译需要自行开启SSH！！！不会请搜索教程！！！
    
默认IP:10.10.10.10

初始密码：password
